P5 Divergence

UN voting ideal points for the five permanent Security Council members, estimated using the Bailey, Strezhnev & Voeten (2017) model. Values are re-centred each year so 0 = the global average across all UN members. Positive = more aligned with the Western liberal bloc; negative = more opposing. The chart shows how the Western P3 (USA, UK, France) and Eastern P2 (Russia/USSR, China) have drifted apart — and occasionally converged — since 1946.
